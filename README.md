# IV1350 – Object-Oriented Design

This repository contains coursework for the **IV1350 Object-Oriented Design** course.

## 📌 Project Overview

The project is based on an **electric bike repair scenario**, where a system is developed to handle the repair process. This includes registering repair requests and managing service steps.

The focus is on applying **object-oriented design principles** to build a structured and maintainable system.

## 🧩 Included Work

- **Seminar 3 – Implementation**  
  Implementation of the system based on the designed architecture, including core functionality and basic testing.

- **Seminar 4 – Exceptions & Design Patterns**  
  Extension of the system with exception handling and the use of design patterns to improve robustness and maintainability.

## 🛠️ Technologies

- Java  
- Object-Oriented Programming (OOP)

## 🎯 Purpose

The purpose of this project is to practice applying established guidelines for **object-oriented design, implementation, and evaluation of software quality**.
